## Summary
Lists documents needed for housing applications.

## Steps
1) Download housing application forms.
2) Gather personal identification.
3) Prepare financial documentation.
4) Upload all required materials.

## Required info / documents
- Government ID
- Proof of enrollment
- Financial forms
- Relationship documentation (if applicable)

## FAQs / Edge cases
- Incomplete applications are often rejected.
- Some documents require notarization.
