## Summary
Outlines the step-by-step housing application process.

## Steps
1) Create housing portal account.
2) Complete application form.
3) Upload required documents.
4) Submit before deadline.
5) Monitor application status.

## Required info / documents
- Login credentials
- Completed application
- Uploaded files

## FAQs / Edge cases
- Late submissions are not accepted.
- Portal outages do not extend deadlines.
